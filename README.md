## Описание
Участие в разработке приложения для управления умными устройствами в доме. В рамках дипломной работы выполнял сбор и формализацию функциональных и нефункциональных требований с помощью изучения интервью, составлял карту пользовательских историй в Miro, выделял MVP, прописывал сценарии и критерии приёмки (Gherkin), составлял диаграмму потоков данных и модель данных (ER Crow’s foots, DFD 3НФ), словарь данных, программу и методику испытаний; разрабатывал прототип приложения в Figma.
